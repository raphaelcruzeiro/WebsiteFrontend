What I use to build things:

I have a quite large set of tools to choose for each task:

The backend of my applications is usually coded in Python using the amazing Django web framework. This powerful combination enables me to deliver high quality software with considerably less development time than the alternatives. Recently, I have also begun rocking some Node backends.

My frontends are created using yeoman which generates me a really nice project structure with Grunt to automate tasks, and Bower for dependency management. The code itself is created using HTML5 ans CSS3 (through SASS) combined with the relativelly new AngularJS Framework. This powerful combination allows me to create an awesome, reliable and maintainable frontend guaranteening an amazing UX.

I have also been known for ocasinally surf the world of mobile development, creating apps for Android and iOS (and sometimes with the same codebase for both platforms using Cordova with Steroids).
